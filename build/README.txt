آیکون‌های این پوشه برای ساخت نرم‌افزار Electron استفاده می‌شوند.

برای جایگزینی آیکون‌ها:
1. فایل icon.ico (256x256) برای ویندوز
2. فایل icon.png (512x512 یا 1024x1024) برای مک و لینوکس

ابزار پیشنهادی برای ساخت آیکون:
- https://www.icoconverter.com/
- https://icon.kitchen/

قالب‌های مورد نیاز:
- Windows: .ico (256x256, 128x128, 64x64, 48x48, 32x32, 16x16)
- macOS: .icns or .png (512x512 minimum, 1024x1024 recommended)
- Linux: .png (512x512 recommended)
