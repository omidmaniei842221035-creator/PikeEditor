🔧 اگر حتماً می‌خواهید روی کامپیوتر نصب کنید:

1نصب PostgreSQL

  1. به postgresql.org بروید
  2. PostgreSQL 14+ نصب کنید
  3. یک database بسازید با نام pos_db

2تنظیم متغیر محیطی

یک فایل .env در پوشه پروژه بسازید:

DATABASE_URL=postgresql://postgres:your_password@localhost:5432/pos_db

3اجرای Migration

npm install
npm run db:push
npm run dev
