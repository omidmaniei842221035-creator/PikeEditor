🔧 حل خطای Build

❌ خطایی که گرفتید:

error during build:
Could not resolve entry module "index.html"

🔍 علت مشکل:

  • تنظیمات Vite برای محیط Replit طراحی شده
  • در Windows نیاز به تغییراتی دارد
  • راه‌حل: استفاده از Development Mode به جای Build

✅ راه‌حل‌های پیشنهادی

1بهترین راه‌حل: Development Mode (بدون Build)

گام 1: نصب Dependencies
npm install
گام 2: اجرای سرور
npm run dev
گام 3: باز کردن مرورگر
http://localhost:5000
✅ مزایا:
  • نیازی به Build ندارد
  • خطا نمی‌دهد
  • همین الان کار می‌کند
  • همه امکانات را دارد
⚠️ نکته: کامپیوتر باید Node.js داشته باشد (که قبلاً نصب کردید)

2راه‌حل دوم: پاک کردن Cache

گاهی مشکل از Cache است:

rmdir /s /q node_modules
del package-lock.json
npm install
npm run dev

3راه‌حل سوم: استفاده از نسخه Web

ساده‌ترین راه: از همان نسخه Web استفاده کنید!

  • ✅ نیازی به نصب ندارد
  • ✅ بدون خطا
  • ✅ همیشه به‌روز
  • ⚠️ نیاز به اینترنت دارد

🎯 خلاصه توصیه

بهترین روش:

npm install
npm run dev

سپس مرورگر را باز کنید: localhost:5000

🐛 اگر باز هم مشکل داشتید

1. چک کردن نسخه Node.js:
node --version

باید 18 یا بالاتر باشد

2. پاک کردن همه چیز و شروع مجدد:
rmdir /s /q node_modules
npm cache clean --force
npm install --force
npm run dev

موفق باشید! 🎉

از Development Mode استفاده کنید - خیلی ساده‌تر است!
