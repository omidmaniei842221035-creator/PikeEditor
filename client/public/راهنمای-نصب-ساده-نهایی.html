🎉 راهنمای نصب نسخه Desktop - نهایی

✅ مشکل حل شد!

حالا می‌تونید نسخه Desktop با دیتابیس SQLite داخلی نصب کنید!

بدون نیاز به PostgreSQL، بدون پیچیدگی، فقط با یک کلیک!

📋 مراحل نصب (خیلی ساده)

1دانلود پروژه

در Replit:

  1. روی منوی ⋮ (3 نقطه) کلیک کنید
  2. "Download as ZIP" را انتخاب کنید
  3. فایل ZIP را Extract کنید (مثلاً در C:\POS-Desktop)

2نصب Node.js (یکبار)

اگر قبلاً نصب کردید، این مرحله را رد کنید.

  1. به nodejs.org بروید
  2. نسخه LTS (18+) را دانلود و نصب کنید

3ساخت نسخه Desktop

Command Prompt را در پوشه پروژه باز کنید:

cd C:\POS-Desktop
npm install
npm run build:desktop

این فرآیند چند دقیقه طول می‌کشد...

4اجرا!

پس از build، به پوشه dist-desktop بروید:

cd dist-desktop

سپس فایل Start-POS.bat را اجرا کنید (دابل کلیک)

یا در Command Prompt:

Start-POS.bat

5باز کردن مرورگر

مرورگر را باز کنید و این آدرس را وارد کنید:

http://localhost:5000
🔐 اطلاعات ورود:
  • نام کاربری: admin
  • رمز عبور: admin123

🎁 ویژگی‌های نسخه Desktop

✅ دیتابیس SQLite داخلی - نیازی به PostgreSQL ندارد
✅ کاملاً آفلاین - بدون نیاز به اینترنت
✅ اجرای ساده - فقط یک فایل bat
✅ قابل انتقال - پوشه dist-desktop را ZIP کنید و به اشتراک بگذارید
✅ تمام امکانات - همان Web version با دیتابیس محلی

📁 مکان دیتابیس

دیتابیس SQLite در این مسیر ذخیره می‌شود:

C:\Users\[نام شما]\AppData\Roaming\POS-System\pos-system.db

📦 اشتراک‌گذاری با دیگران

می‌توانید پوشه dist-desktop را:

  1. ZIP کنید
  2. روی فلش کپی کنید
  3. با دیگران به اشتراک بگذارید
⚠️ نکته: کامپیوتر مقصد باید Node.js داشته باشد. اولین بار که Start-POS.bat را اجرا می‌کنید، dependency ها نصب می‌شوند.

🐛 حل مشکلات

مشکل: npm install خطا می‌دهد

npm cache clean --force
npm install --force

مشکل: Port 5000 اشغال است

فایل Start-POS.bat را ویرایش کنید و قبل از node server.js اضافه کنید:

set PORT=8080
