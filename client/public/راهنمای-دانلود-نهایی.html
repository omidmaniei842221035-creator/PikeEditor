گزینه 2: نصب روی کامپیوتر شخصی (پیچیده)

⚠️ هشدار: این روش نیاز به دانش فنی دارد!

📥 مرحله 1: نصب پیش‌نیازها

الف) نصب Node.js:
  1. به سایت nodejs.org بروید
  2. نسخه LTS (18 یا بالاتر) را دانلود کنید
  3. نصب کنید (Next > Next > Install)
  4. Command Prompt را باز کنید و تست کنید: node --version
ب) نصب Build Tools (اختیاری اما توصیه می‌شود):
  1. Command Prompt را به عنوان Administrator باز کنید
  2. اجرا کنید: npm install --global windows-build-tools
  3. صبر کنید تا نصب کامل شود (ممکن است چند دقیقه طول بکشد)

📦 مرحله 2: دانلود کد پروژه

روش دانلود:
  1. در Replit، روی منوی ⋮ (3 نقطه) کلیک کنید
  2. گزینه "Download as ZIP" را انتخاب کنید
  3. فایل ZIP را در یک پوشه دلخواه Extract کنید

🔧 مرحله 3: نصب و اجرا

دستورات (در Command Prompt):
  1. به پوشه پروژه بروید: cd C:\path\to\project
  2. نصب کتابخانه‌ها: npm install
  3. ساخت پروژه: npm run build
  4. اجرا: node dist\index.js
  5. مرورگر را باز کنید: http://localhost:5000
⚠️ حل مشکلات احتمالی:
  • خطا در npm install: نصب Build Tools (مرحله 1-ب) الزامی است
  • Port 5000 اشغال: از set PORT=8080 استفاده کنید
  • خطای Python: Python 3 نصب کنید از python.org
