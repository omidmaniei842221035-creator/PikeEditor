# راهنمای ساخت و نصب نرم‌افزار دسکتاپی 🖥️

این راهنما نحوه ساخت فایل نصبی `.exe` برای نسخه دسکتاپ سامانه مانیتورینگ هوشمند پایانه‌های فروشگاهی را توضیح می‌دهد.

## ⚠️ نکته مهم

برای ساخت فایل `.exe` نصبی، **باید این پروژه را روی کامپیوتر ویندوزی خودتان اجرا کنید**. در محیط Replit امکان ساخت installer ویندوزی وجود ندارد.

## 📋 پیش‌نیازها

### برای ساخت installer (روی کامپیوتر ویندوزی):

1. **کامپیوتر با ویندوز 10 یا بالاتر**

2. **Node.js** (نسخه 18 یا بالاتر)
   - دانلود از: https://nodejs.org/
   - برای بررسی نصب: در CMD بزنید `node --version`

3. **Git** (برای کلون کردن پروژه)
   - دانلود از: https://git-scm.com/
   - یا می‌توانید پروژه را به صورت ZIP دانلود کنید

## 🚀 مراحل ساخت نرم‌افزار دسکتاپی (روی کامپیوتر ویندوزی)

### مرحله 1: دانلود پروژه

**روش 1: با Git (توصیه می‌شود)**
```bash
# کلون کردن پروژه
git clone <آدرس-ریپازیتوری-پروژه>
cd <نام-پوشه-پروژه>
```

**روش 2: دانلود ZIP**
1. از Replit یا GitHub، پروژه را به صورت ZIP دانلود کنید
2. فایل ZIP را Extract کنید
3. پوشه پروژه را باز کنید

### مرحله 2: نصب وابستگی‌ها

در پوشه پروژه، CMD یا PowerShell را باز کنید و دستور زیر را اجرا کنید:

```bash
npm install
```

این کار ممکن است 2-5 دقیقه طول بکشد (بسته به سرعت اینترنت).

### مرحله 3: ساخت فایل نصبی `.exe`

```bash
npm run electron:build:win
```

⏱️ **این مرحله ممکن است 5-10 دقیقه طول بکشد.**

این دستور به ترتیب:
1. ✅ فایل‌های React را build می‌کند (frontend)
2. ✅ سرور Express را bundle می‌کند (backend)  
3. ✅ فایل‌های Electron را compile می‌کند
4. ✅ دیتابیس SQLite را آماده می‌کند
5. ✅ یک installer ویندوزی (.exe) می‌سازد

### مرحله 4: یافتن فایل نصبی

پس از اتمام موفقیت‌آمیز build، فایل نصبی در مسیر زیر قرار دارد:

```
release/
  └── سامانه مانیتورینگ POS-1.0.0-Setup.exe
```

**این فایل را می‌توانید:**
- ✅ روی همان کامپیوتر نصب کنید
- ✅ روی فلش یا هارد کپی کنید
- ✅ به دیگران بدهید
- ✅ روی چند کامپیوتر نصب کنید

## 💾 ویژگی‌های نسخه دسکتاپی

### دیتابیس محلی (SQLite)
- ✅ تمام داده‌ها به صورت محلی ذخیره می‌شوند
- ✅ نیازی به اتصال به اینترنت نیست
- ✅ فایل دیتابیس در مسیر زیر قرار دارد:
  ```
  %APPDATA%/سامانه مانیتورینگ POS/pos-system.db
  ```

### قابلیت‌های نصب
- ✅ نصب در پوشه دلخواه
- ✅ ایجاد شورتکات روی دسکتاپ
- ✅ اضافه شدن به Start Menu

## 🔧 دستورات توسعه

### اجرای نسخه توسعه Electron

```bash
# اجرای وب‌سرور (در یک ترمینال)
npm run dev

# اجرای Electron (در ترمینال دیگر)
npm run electron:dev
```

### ساخت بدون installer (برای تست)

```bash
npm run pack
```

این دستور فایل‌های برنامه را بدون ساخت installer آماده می‌کند.

## 📦 سایر پلتفرم‌ها

### ساخت برای macOS

```bash
npm run electron:build -- --mac
```

### ساخت برای Linux

```bash
npm run electron:build -- --linux
```

## 🎨 سفارشی‌سازی

### تغییر آیکون برنامه

1. یک فایل آیکون `.ico` (256x256 پیکسل) آماده کنید
2. فایل را در پوشه `build/` با نام `icon.ico` قرار دهید
3. در `package.json`، خط زیر را اضافه کنید:

```json
"win": {
  "icon": "build/icon.ico",
  ...
}
```

### تغییر نام نرم‌افزار

در `package.json`:

```json
{
  "name": "pos-monitoring-system",
  "productName": "نام دلخواه شما",
  "description": "توضیحات دلخواه",
  ...
}
```

## ⚠️ نکات مهم

### محدودیت‌ها

1. **فایل‌های بزرگ**: نرم‌افزار نهایی حدود 150-200 مگابایت خواهد بود
2. **ویندوز 10+**: نیاز به ویندوز 10 یا بالاتر دارد
3. **حافظه**: حداقل 4GB RAM توصیه می‌شود

### رفع مشکلات رایج

#### خطا در build:
```bash
# پاک کردن فایل‌های قبلی
rm -rf dist dist-electron release
npm run electron:build:win
```

#### نرم‌افزار باز نمی‌شود:
- از ویندوز Defender یا آنتی‌ویروس استثنا بگیرید
- با دسترسی Administrator اجرا کنید

#### دیتابیس خالی است:
- از بخش بازیابی فایل بک‌آپ را import کنید
- یا داده‌های جدید وارد کنید
