# POS Monitoring System - Windows Installation

## Requirements
- Windows 10/11 (64-bit)
- Node.js 18 or 20 (https://nodejs.org)

## Build Instructions

1. Extract the zip file
2. Open Command Prompt in the folder
3. Run: `BUILD.bat`
4. Wait 5-10 minutes
5. Find exe in `release` folder

## Troubleshooting

**If npm install fails:**
- Check internet connection
- Run as Administrator

**If build fails:**
- Delete `node_modules` folder
- Run `BUILD.bat` again

## Usage

Double-click the exe file to run.
Data is saved in: `%APPDATA%\سامانه مانیتورینگ POS\`
